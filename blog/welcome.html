Welcome to my blog

Ziyun Zhao · 6 July 2026 · 3 min read

Hello, and welcome! This is the very first post on my new site. I built this space to think out loud — to keep notes on what I'm learning, share the things I'm building, and occasionally write about ideas that don't fit anywhere else.

If you're reading this, thanks for stopping by. Here's a little about what to expect.

What I'll write about

I don't want to box myself in too tightly, but a few themes I expect to keep coming back to:

  • Building things — projects, experiments, and the lessons that only show up once you actually ship.

  • Learning in public — notes on topics I'm working through, written for the version of me who didn't understand them yet.

  • Reflections — the occasional longer piece on work, habits, and how I think about things.

How this site is built

This site is a set of plain HTML pages styled with the Clarity template — no build step, no database, nothing to maintain. Each blog post is its own file in the blog/ folder, and the homepage links to them.

Writing a new post is deliberately boring: copy the template, change the words, add one link on the homepage. For example, a code block looks like this: 

def greet(name):
    return f"Hello, {name}! Welcome to the blog."

print(greet("world"))

That's really all there is to it. More soon.
